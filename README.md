**<u>Installation:</u>**



**WARNING: This will delete your current config files!**

```shell
cd ~
```

```shell
git clone https://github.com/jebmu/dotfiles.git
```

```shell
cd dotfiles
```

```shell
chmod +x symlinks.sh
```

```shell
./symlinks.sh
```
